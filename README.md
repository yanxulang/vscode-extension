# 言序语言 · VS Code

言序编程语言的官方 VS Code 扩展。

## 功能

- `.yx` 文卷语法高亮；
- 中文引号、括号自动配对与代码折叠；
- 变量、判断、循环、函数、类和模块代码片段；
- 编辑器右上角一键运行当前文卷；
- 命令面板打开言序 REPL 或在线文档。

## 使用

请先[安装言序命令行](https://yanxu.dev/download/)，然后打开任意 `.yx` 文件。默认会从 `PATH` 查找 `yanxu`；若安装在其他位置，请设置 `yanxu.executablePath`。

# 变更记录

## 未发布

暂无。

## 1.4.0

- 同步言序 1.1.7、原生 ABI v2 与机器可读能力握手。
- LSP 补全锁定包的公开成员签名，言窗 API 无需编辑器内置特例。
- 新增言窗应用/事件片段和发布版 `--bundle` 工作区任务。
- 离线标准库索引由言序 1.1.7 CLI 重新生成。

## 1.3.2

- 同步言序核心 1.1.6，并由该版本的机器 API 重新生成 25 模块离线标准库快照。
- 标准库快照新增核心版本标识，扩展构建会拒绝模块数量或核心版本不一致的快照。
- 文档同步 YXB 权限收紧、言包 0.2.1 独立自包含 Release 和核心 CLI 安装边界。

## 1.3.1

- 修正命令行参数片段为`环境.参数()`，并新增使用`进程.执行()`的受限子进程片段，与言序 1.1.5 机器 API 清单保持一致。

## 1.3.0

- 同步言序核心 1.1.5：离线标准库快照增至 25 个模块，新增`进程`、`资源`及相关代码片段。
- 包导入索引支持格式 2 根依赖别名、显式导出和包子模块，并阻止应用直接穿透访问未声明的传递依赖。
- 新增“编译工作区 YXB 应用”命令与任务，直接调用`yanxu compile`生成不依赖原始源码的应用归档。
- 补充包公开模块、嵌入资源和命令行参数片段，扩展结构检查会验证 1.1.5 标准库快照完整性。

## 1.2.0

- 新增扩展侧工作区符号索引，覆盖类、协议、字段、方法、函数、参数、变量、常量和导入别名，并增量跟踪未保存文卷。
- 递归解析相对路径和`包:`导入并索引工作区外模块；包索引只读使用已有清单、锁文件与缓存，跨模块补全、悬停、签名提示和定义跳转遵守`公`有导出边界。
- 通过当前`yanxu 标准库 --json`动态索引标准库，附带 23 个模块的离线快照与只读虚拟定义文卷。
- 代码补全和工作区符号查询支持无声调全拼与拼音首字母；语言服务返回的补全项也会自动补充拼音筛选键。
- 新增“重建符号索引”命令和索引范围、外部模块、拼音补全配置，加入完整解析、作用域、继承、导入和拼音测试。

## 1.1.4

- 新增`字节串`类型高亮与回退补全，标准模块补全增加`字节`并增至 23 个。
- 新增字节构造、二进制 HTTP 请求和带 EOF/半关闭/TCP 无延迟的字节流片段。
- 扩展版本、说明和测试同步言序核心 1.1.4；核心 LSP 可从版本化 API 清单补全标准库成员。

## 1.1.3

- 网络与套接字说明同步逐跳 HTTP 权限、DNS 后真实 IP 校验、调用级超时、UTF-8 分片缓冲和资源上限。
- TCP 服务端、UDP 绑定片段标明独立包权限；扩展版本同步言序核心 1.1.3。
- GitHub Actions 依赖固定到已验证的完整提交 SHA。

## 1.1.2

- 套接字加入标准模块与类型补全，新增 TCP、UDP 和`SOCKET_*`错误捕获片段。
- TextMate 类型高亮、说明、测试与扩展版本同步言序核心 1.1.2；标准模块补全增至 22 个。

## 1.1.1

- 高亮、回退补全与悬停同步`父`和`是`，支持父类实现调用与原生类型判断。
- 继承片段补齐`父.初始化/父.方法`，新增类型判断、带超时/响应上限的 HTTPS 请求和`NET_*`错误捕获片段。
- 扩展版本和说明同步至言序核心 1.1.1，并保持 21 个稳定标准模块补全。

## 1.1.0

- 标准模块路径补全与导入选择同步到 21 个模块，新增`Base64`、`正则`、`URL`和`日期`。
- 新增四个 1.1 模块的可运行代码片段；语义 LSP 由言序 1.1 核心提供成员补全、悬停和类型诊断。
- 扩展版本和说明同步至言序核心 1.1.0。

## 1.0.1

- 将扩展与 Marketplace 的品牌图标更新为与官网、文档站统一的朱红底白色“言”字图标。
- 标准模块补全与导入片段新增`随机`、`标识`、`模板`和`校验`，与核心发行后标准库保持同步。

## 1.0.0

- 同步言序 1.0 的`异`、`候`、`任务<T>`、取消、状态与结构化`并候`语法、补全、悬停和代码片段。
- 新增“迁移当前文卷”命令和 migrate 任务，调用核心`yanxu 迁 --写`。
- 标准模块补全扩展为 13 个正式模块，并与 1.0 TextMate/Tree-sitter 语法对齐。

## 0.7.1

- 加入蓝紫色“言”字代码括号图标，并保留 SVG 设计母版。
- 完善块结构、`否则`、`救`、协议与文档注释的智能缩进规则。
- 在语言服务不可用时提供关键字、类型、常量、内建函数与标准模块补全。
- 加入内建函数参数签名提示，以及关键字、类型和内建函数的悬停说明。
- 注册`yanxu 调试服务` DAP 适配器，支持断点、继续、单步、调用栈和作用域变量。
- 加入一键迁移当前文卷命令，调用 `yanxu 迁 --写` 更新弃用语法。
- 高亮与片段新增协议、字段、泛型、异步函数、任务等待及结构化并发语法。
- 标准库导入片段新增 13 个模块的选择列表，包括路径、环境、哈希、编码、统计与 CSV。

## 0.7.0

- 自动启动 `yanxu 语言服务`，提供实时诊断与文档格式化。
- 加入静态检查、字节码 VM、踪迹与工作区测试命令。
- 加入可复用 REPL，可直接运行选区或当前行。
- 加入语言服务状态栏、重启、启停和通信日志入口。
- 加入自动发现的言序 VS Code 任务与跨平台参数传递。
- 语言服务不可用时安全降级，其他编辑与运行功能仍可使用。
- 加入单元测试、CI 与标签驱动的 GitHub/Marketplace 发布流程。
- 高亮与代码片段同步`公`、`承`、联合类型与 0.7 容器/数学原语。

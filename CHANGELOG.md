# 变更记录

## 0.7.0

- 自动启动 `yanxu 语言服务`，提供实时诊断与文档格式化。
- 加入静态检查、字节码 VM、踪迹与工作区测试命令。
- 加入可复用 REPL，可直接运行选区或当前行。
- 加入语言服务状态栏、重启、启停和通信日志入口。
- 加入自动发现的言序 VS Code 任务与跨平台参数传递。
- 语言服务不可用时安全降级，其他编辑与运行功能仍可使用。
- 加入单元测试、CI 与标签驱动的 GitHub/Marketplace 发布流程。
- 高亮与代码片段同步`公`、`承`、联合类型与 0.7 容器/数学原语。

## 0.3.0（预发布）

- 高亮新增`元`类型。
- 加入元组与半开切片代码片段。
- 与言序 `0.3.0-alpha.2` 语法同步。

## 0.3.0-alpha.1

- 首次发布官方言序语法高亮。
- 加入自动配对、折叠与常用代码片段。
- 加入运行当前文卷、打开 REPL 和在线文档命令。
- 高亮列、典、迭代、不可改写绑定与结构化错误语法。
